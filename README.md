Download music from YouTube and edit tags

## Install
Install ffmpeg: 
```
$ sudo apt-get install libav-tools
```

## Deploy 
Add buildpack to Heroku: 
```
$ heroku buildpacks:add https://github.com/jonathanong/heroku-buildpack-ffmpeg-latest
```
